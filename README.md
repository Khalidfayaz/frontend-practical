# People Management Dashboard

A responsive React.js dashboard for managing people/employees with a clean, modern UI.

## Tech Stack

- **React.js** - Frontend library
- **CSS3** - Styling and responsive design
- **Lucide React** - Icons (optional, can be added for enhanced icons)

## Features

- Clean, pixel-perfect UI matching the provided design
- Responsive design for desktop and mobile devices
- Search functionality by employee name or role
- Interactive hover effects on cards
- Pagination controls
- Collapsible sidebar for mobile view
